Usage Guidelines

    Modals should be used as a way to get the user to enter information in quickly before continuing.

    Do not suddenly open modals without having the user interact with the page.

    Darken the page behind the modal window.

    Allow the user a clear path to close the modal. Most of the time, the ability to click outside of the modal box to close is acceptable. The exception to this is in the case of a confirmation message.

    Multiple modals should not open on top of each other. Multiple step modals are also not good practice.
