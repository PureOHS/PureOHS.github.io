Usage Guidelines

    Each column div class needs to follow this pattern: col-sm-x col-md-x. This will ensure appropriate responsiveness on tablet devices.

    Always use the .container fixed width class. The .container-fluid class is not used in PureOHS.

Sample Code

<div ul-auto-height>
     <ul-grid source="{{ 'main.test.person.grid' | ulApiUrl : { sort: $ulGrid.params.sort, skip: $ulGrid.params.skip, top: $ulGrid.params.top} }}" paging="">
         <columns key="ObjectId">
             <column field="GivenName" width="170" header="First Name"></column>
             <column field="FamilyName" header="Last Name" sort-priority="0" sort-direction="asc"></column>
             <column field="EmployeeID" pin="right">
                 <a href="{{ '/employee/' + $it.EmployeeID }}">{{ $it.EmployeeID }}</a>
             </column>
             <column field="GovernmentID" sortable="false" pin="left"></column>
             <column field="DateOfBirth"></column>
             <column field="CompanyName"></column>
             <column field="IsActive" header="Active">
                 <input type="checkbox" ng-model="$it.IsActive" disabled="" />
             </column>
         </columns>
     </ul-grid>
</div>

Usage Guidelines

    Each grid should have a clickable link to dive into the CRUD aspects of the work section.

    All action items should be housed in a pinned right-side column. Icon links are preferrable in those sections.

    Column headers that are sorted should have a caret indicator next to the text.

    Column headers color is color: #9B9B9B and non-link text color is color: #222.

Sample Code

<div class="container">
 <div class="row">
   <div class="col-sm-12 col-md-12">
     <div class="panel panel-default">
     <div class="panel-body">
     Panel content here
  </div>
 </div>
</div>

Usage Guidelines

    Panel headers should sit above the panel as an h3 (as shown above). Any additional headers should be inside the panel as an h4.

    Do not capitalize in title case except as the first word: "for", "the", "a", "an", "in", "of", "on".

    Each panel should have this styling applied: box-shadow: 0px 0px 3px 0px #AAAAAA.
